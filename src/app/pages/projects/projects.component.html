Proyectos Desarrollados

EveryMed Admin

Proyecto Propio Frontend

Aplicación web progresiva (PWA) para gestión médica desarrollada con Angular 20. Plataforma completa para administración de clínicas, expedientes médicos, citas y más.

Características principales:

  • Progressive Web App (PWA) con soporte offline
  • Autenticación JWT y sistema de roles (admin, doctor, secretary, pharmaceutical)
  • Módulos: Dashboard, Agenda, Expedientes, Finanzas, Farmacia, Inventario
  • Expedientes médicos conforme a NOM-024-SSA3
  • Notificaciones en tiempo real y gestión de citas
  • Despliegue nativo con Capacitor (Android/iOS)

Stack tecnológico:

Angular 20 TypeScript Tailwind CSS 4 Capacitor 7 PWA Signals

Arquitectura: Standalone Components, OnPush Change Detection, Lazy Loading

EveryMed API

Proyecto Propio Backend

API REST robusta desarrollada con .NET 9 siguiendo Clean Architecture. Proporciona servicios esenciales para la gestión médica con autenticación JWT, gestión de expedientes y más.

Características principales:

  • Arquitectura limpia (Clean Architecture) con separación de capas
  • Autenticación JWT con refresh tokens y políticas de autorización
  • Módulos: Auth, Users, Patients, Medical Records, Appointments, Pharmacy
  • Entity Framework Core con SQL Server
  • Notificaciones y recordatorios automáticos
  • Swagger/OpenAPI para documentación interactiva

Stack tecnológico:

.NET 9 ASP.NET Core Entity Framework SQL Server JWT Bearer Swagger

Arquitectura: Domain, Application, Infrastructure, Presentation layers

Sobre EveryMed

EveryMed es una plataforma completa de gestión médica diseñada para médicos individuales y clínicas. El sistema integra un frontend moderno (EveryMed Admin) con una API robusta (EveryMed API) para proporcionar una solución integral en el sector de la salud.

Funcionalidades del sistema:

  • Gestión de expedientes médicos digitales
  • Sistema de citas y agenda médica
  • Módulo de farmacia con inventario
  • Finanzas y facturación

Tecnologías integradas:

  • Comunicación REST API con JWT
  • Base de datos relacional SQL Server
  • Notificaciones en tiempo real
  • Servicio de correo electrónico (MailKit)
